# TrackCar Mobile App 요구사항 및 기능 명세서

- 작성일: 2026-03-31
- 버전: v2.0
- 상태: 재정비 초안

---

## 1. 문서 목적

본 문서는 TrackCar Mobile App의 **MVP 기능 범위, 역할, 메뉴 구조, 핵심 운영 시나리오**를 정의한다.

기준:
- Internal Admin Web는 **설치/개통/검증** 전용 내부 시스템이다.
- Mobile App은 고객사 사용자를 위한 **운영/관제 Self-Service 채널**이다.
- 본 문서는 현재 backend 구현과 향후 모바일 개발 방향을 함께 고려한 **설계 기준 문서**다.

---

## 2. 설계 원칙

1. 모바일 앱은 **운영 중 사용 경험**에 집중한다.
2. 설치/개통/연동 검증은 Internal Admin Web가 담당한다.
3. 모바일 홈은 복잡한 운영 기능을 넣지 않고 **상태 요약 + 진입 버튼**에 집중한다.
4. 운영 기능은 별도 **운영 메뉴** 아래로 모은다.
5. 차량 지도 관제는 목록과 분리된 **전체차량위치 화면**으로 제공한다.
6. 알림 관리와 알림 이력 조회를 구분한다.

---

## 3. 사용자 역할

| 역할 | 설명 | 권한 요약 |
|------|------|-----------|
| OWNER | 고객사 관리자 | 차량/운행/알림 조회 + 운영 메뉴(담당자/그룹/알림 관리) 접근 |
| STAFF | 고객사 담당자 | 차량/운행/알림 조회 중심, 운영 메뉴 접근 제한 |
| Internal Admin | 설치기사/운영자/검증담당자 | Mobile App 범위 제외 |

### 3.1 OWNER 권한
- 홈 대시보드 조회
- 전체차량위치 조회
- 차량 목록/상세 조회
- 운행 이력 조회
- 알림 센터 조회
- 담당자 관리
- 그룹 관리
- 알림 관리
- 내 정보 / 내 설정 조회

### 3.2 STAFF 권한
- 홈 대시보드 조회
- 전체차량위치 조회
- 차량 목록/상세 조회
- 운행 이력 조회
- 알림 센터 조회
- 내 정보 / 내 설정 조회

### 3.3 STAFF 제한
- 담당자 관리 불가
- 그룹 관리 불가
- 알림 관리 불가

## 4. MVP 핵심 기능 범위

### 4.1 홈 대시보드
표시 항목:
- 전체차량
- 운행중
- 정차중
- 오프라인

액션:
- 전체차량위치 버튼

제외:
- 조치 필요 차량 리스트
- 최근 알림 리스트
- 빠른 이동 링크

### 4.2 전체차량위치
목적:
- 조직 내 차량 전체를 지도에서 확인

기능:
- 차량 marker 표시
- 차량 상태별 marker 구분
- 차량번호/기사명 검색
- 그룹 필터
- 차량 선택 시 차량 상세 이동

### 4.3 차량
기능:
- 차량 목록 조회
- 차량 상세 조회
- 상태 뱃지 확인
- 현재 위치 확인

### 4.4 운행 이력
기능:
- 운행 이력 목록 조회
- 운행 상세 조회
- 경로 지도 표시

### 4.5 알림 센터
기능:
- 알림 목록 조회
- 심각도/상태 필터
- 알림 상세 조회
- 읽음 처리

### 4.6 운영 메뉴 (OWNER)
하위 기능:
- 담당자 관리
- 그룹 관리
- 알림 관리

### 4.7 내 정보
기능:
- 사용자 정보 조회
- 알림 설정 조회/변경

---

## 5. 운영 메뉴 정의

### 5.1 담당자 관리
목적:
- 고객사 소속 담당자(STAFF) 계정 관리

기능:
- 목록 조회
- 추가
- 수정
- 활성/비활성

### 5.2 그룹 관리
목적:
- 고객사 운영 단위 그룹 관리

기능:
- 목록 조회
- 추가
- 수정
- 삭제

### 5.3 알림 관리
목적:
- 차량별 알림 수신 설정 관리

현재 포함 기능:
- 시동 ON/OFF 알림
- 공회전 알림

추후 기능:
- 위치 알림 (후속 설계)

설정 방식:
- 차량 선택
- 알림 종류별 on/off
- 저장

## 8. 데이터 범위 원칙

1. OWNER는 고객사 전체 기준 조회 가능
2. STAFF는 허용된 범위만 조회 가능
3. 차량 관제는 최신 telemetry 기준으로 상태를 판단
4. Mobile App은 설치/개통이 아니라 **운영 중 상태 확인**에 집중

## 9. Mobile / Admin Web 경계

### Internal Admin Web 담당
- 고객사 생성
- DTG 장치 등록
- 차량 등록

- 매핑

### Mobile App 담당
- 운영 상태 조회
- 차량/운행/알림 확인
- 담당자/그룹/알림 설정 관리
- 개인 설정
